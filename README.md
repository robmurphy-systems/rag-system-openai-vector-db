# RAG System: OpenAI + Vector Database
End-to-end RAG pipeline using LLMs and vector databases for retrieval, grounding, and response generation.

## What This Demonstrates
- End-to-end system design (ingestion → retrieval → generation)
- Practical application of LLMs with vector search
- Understanding of retrieval quality and grounding techniques

## Pipeline Overview
Document → Chunking → Embeddings → Vector DB → Retrieval → LLM Response
